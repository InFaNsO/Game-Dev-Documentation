### Project context

- **Development strategy — ship one pillar first, then carry forward.** The dream game is a four-pillar fusion: roguelike storytelling + city builder + automation + looter-shooter/dungeon-crawler (`Main Game/01 Concept & Idea.md`, `04 Game Pitch.md`). Plan: build the hardest, most self-contained pillar — the Looter Shooter — as its own polished standalone game first, then fold its tech into the main game with **additive changes only, no rewrites** (`Looter Shooter/10 Unity Code Architecture.md` §22 Carry-Forward Checklist). Rationale: procgen room assembly + the Sparks of Hope × XCOM × Expedition 33 tactical combat model are the riskiest unknowns; the standalone de-risks them and ships a real product.

- **Locked design — DREAM GAME vision (task 5, Phase A — locked 2026-06-06; user's own concept, full pitch in `Main Game/04 Game Pitch.md`).** Set in the **Mani world**, a *different era* from the Looter Shooter, **no plot connection** (callbacks only). **Concept:** an **accidentally-immortal** ruler expands his kingdom into a **Bermuda-Triangle-like zone**, colonizes a mysterious island riddled with **Solo-Leveling-style portal dungeons** (beasts + otherworldly era-advancing tech). He runs **one persistent Main City** + a **Satellite island City** (Anno-style city-building w/ multi-town logistics). **Roguelike engine:** fail to clear a dungeon in time → its beasts **break out** and destroy the island → **re-animate in the Main City** → redo, bigger (persistent tech survives). Two answers per open dungeon: **clear it** (offense → the *only* source of era-tech) or **defend the island** (survival). **Four pillars + interaction:** looter-shooter = the delves (loot = power + city advancement, reuses LS combat); automation = **Mani spell-engineering inside gems** (Modulus/particle-system); roguelike = re-animation + dungeon-delivered story + the Triangle mystery; city-building = Anno needs + Main↔Satellite logistics. **Callbacks:** portals = interdimensional gates (Breach-echo); immortality = the Husk-making force with mind intact; otherworldly = Akashic. **DREAM-GAME INTERNAL DECISIONS — DEFERRED to dream-game DEV time (NOT now):** the dungeon-break clock (keystone — juggling ticking gates), Tarkov-style extract-before-fall, clear-vs-defend balance (clearing = only era-tech path), keeping the Main City engaging, run granularity. Rationale: nothing imminent depends on them, and each gets a *far* better answer from a playable build than from guessing on paper years ahead (same discipline as the combat "defer numbers to build-time"). The dream game is built LAST (see the build-ladder roadmap below).
