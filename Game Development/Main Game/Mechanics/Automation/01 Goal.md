# Automation — Goal

### Main goal for the player

Build the Accord frontier's finest **spell-gear workshop** — master **Conjuration Engineering**, scale production, and **out-compete the rival workshops** — riding the frontier boom all the way to the Accord's forbidden obsession: the **"new Mani"** that never happens.

Concretely, the player:
- **Designs** spells as processes on the gem grid (Emit → Shape → Drive; *you build the machine that makes the lance* — form = function = visual).
- **Climbs the element matrix** — 4 base → 6 pairs → 4 triples (14 craftable; the all-4 cell = Akash, visible and forbidden all game).
- **Produces** — runs the warehouse factory, feeds the mega assembler; **throughput = market output**.
- **Forks every finished good** — **SELL** (money now) or **ANALYZE** (Research Points later) — the core economic choice.
- **Competes** — a living market, 2–4 rival workshops, head-to-head bake-off contracts.
- **Grows on three axes** — capability (RP/tech), quality (Surface A: gems + design), quantity (Surface B: throughput).

### What the player is NOT doing

- **Not** fighting — the Test Room is the only firing range; no combat gameplay (the spell *data* feeds the combat pillar).
- **Not** refining gems — bought pre-refined; refining is the Looter Shooter's minigame.
- **Not** managing workers — fully machine-automated; no staff sim.
- **Not** serving customers — selling = market screen + written commissions; customers appear only as ambient set-dressing.
- **Not** running shipping networks — one warehouse; multi-site logistics is the city pillar's job.
