# Automation — Game Loop

> Two interlocking loops — the **design loop** (Surface A, the signature) and the **production & market loop** (Surface B, the engine) — plus the progression loop that arcs across a playthrough.

---

## Design loop — engineer a spell (Surface A, the signature)

`spot a market gap or commission → pick gem(s) + gear housing → build the process on the gem grid (Emit → Shape → Drive / Split / Merge) → fire it in the Test Room → read the derived stats (damage · damage-type · delivery · cast time) → iterate → finalize the blueprint (= bill of materials)`

- The form comes from shapers, never hand-placement — *you build the machine that makes lances.*
- Cast time is derived from the process's flow rate → engineering efficiency is combat-legible.
- The Test Room is dopamine + debugger + design bench in one.

## Production & market loop — run the shop (Surface B, the engine)

`procure (gems · generic raws · per-element raws) → fabricate generic + specialized parts → route across the warehouse to the mega assembler → finished spell-gear → FORK: sell (money) or analyze (RP) → market prices shift with supply + rival activity → opt into bake-off contracts → reinvest (machines · gems · expansion)`

- The mega assembler visibly builds each product — and stalls if any BOM part runs dry.
- Specialize deep (dominate one element's segment) vs spread wide (flexible, never cheapest).
- Rivals flood, undercut, and bid against you — the pressure that keeps B strategic.

## Progression loop — climb the frontier

`analyze finished goods → RP → unlock matrix cells (recipes) + Conjuration ops + Workshop/Expansion tech → reputation rises with success → market widens (new elements · bigger/purer gems · higher-slot gear · richer contracts) → multi-Mani spells → the endgame Akash-tease contracts`

- Capability compounds: better craft → higher-grade analysis → faster research.
- The element matrix doubles as the research UI — the forbidden all-4 cell taunts you to the end.

---
